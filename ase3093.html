Course Info.

Course descriptions

  • Undergraduate entry level control course covering dynamic system modeling, Laplace transform and transfer functions, frequency domain analysis, state space descriptions, classical control design techniques, stability margins, and so on.

Course contents and reference notes

  1. Feedback and closed-loop systems

  2. Dynamics review

  3. Dynamic modeling of mechanical systems (translation, rotation)

  4. Dynamic modeling and linearization

  5. Linear dynamical systems and Laplace transform

  6. Describing linear dynamical systems

  7. Impulse response and frequency response

  8. Time domain responses and stability

  9. Block diagrams and feedback interconnection

  10. Block diagram algebra

  11. Poles and eigenvalues

  12. Steady state error and system types

  13. Steady state error rejection

  14. Effects of pole locations on time domain responses

  15. Root locus and design examples

  16. Frequency domain analysis

  17. Bode plots

  18. Stability margin
